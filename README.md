# caixa-berghem
Aplicação web feita em Django de MongoDB (noSQL) para a oportunidade de emprego na empresa Berghem CyberSecurity

---------------

## Instalação e Execução da Aplicação

1. É necessário ter o git no seu computador (Se não tiver o Git, baixe aqui: https://git-scm.com/downloads).

2. No terminal, rode o comando "Git clone https://github.com/Jonntz/caixa-berghem".
Ps: Sem as aspas.

3. Entre na pasta do projeto digitando "cd caixa-berghem".
Ps: Sem as aspas.

4. Ative a Virtual Env com o comando "source venv/bin/activate".
Ps: Sem as aspas.

5. Agora basta executar a aplicação digitando "python3 manage.py runserver"
Ps: Sem as aspas.

DETALHE!!!!
Caso o passo 5 dê algum conflito, verifiqque se o python está instalado na sua máquina ou rode simplesmente "python manage.py runserver".

-----------

## Utilizando a aplicação

Com o servidor rodando basta digitar o endereço a seguir no seu navegador:
http://127.0.0.1:8000/
